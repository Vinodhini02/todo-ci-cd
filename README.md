# Todo Flask App

A simple *Todo* web application built with *Flask*, Dockerized, and ready for CI/CD.

---

## Features

- Add, update, delete, and view tasks
- REST API endpoints
- Dockerized for easy deployment
- Automated testing with Pytest

---

## Tech Stack

- Python & Flask
- Docker
- Pytest
- CI/CD with GitHub Actions
